Detailed Description

ChessGuiBoard is a data structure for manipulating a 2d array of ChessGuiBoardCell.

It's functionality is completely wrapped inside of the ChessGui class and so never needs to be accessed directly by the student

Note:
Depends heavily on cs240chess.glade
