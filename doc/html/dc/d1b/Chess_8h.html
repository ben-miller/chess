#define LOG_LEVEL_HIDE_MASK   (0)

Set the log levels (using the Glib log levels) that will be ignored by the Chess logging function.

This mask will hide messages from being displayed in the ChessGui message box, it will not prevent them from being sent to the command line when not using the GUI. Set this flag at compile time using the compiler directive -D and OR-ed values of the Glib log levels. For example: 

   g++ -g -o Chess.o (options) -DLOG_LEVEL_HIDE_MASK=(G_LOG_LEVEL_DEBUG|G_LOG_LEVEL_INFO) Chess.cpp

will prevent debug and info messages from being displayed.
