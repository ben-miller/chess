Detailed Description

This class is the main interface between the Students code and the provided GUI.

The Student will implement each of the signal handlers found in this class. It is permitted to add member data and functions. The constructor and the destructor should be finished to suit the needs of the student.

Note:
Access the convenience functions using the pointer Chess::gui
