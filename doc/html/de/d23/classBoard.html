Detailed Description

An interface for the Board class.

This class represents a chess board whose 64 cells (and therefore pieces) can be acessed via row & column indeces

Constructor & Destructor Documentation

Board::Board ( )

Allocates an 8x8 array of Cell pointers and the respective Cells on the heap Each of these Cells is initialized to point to emptyPiece.

Note:
IMPORTANT: This is only part of the board's initialization. After declaring a Pieces object, we need to call BindCellsToPieces to finish setting the board.
